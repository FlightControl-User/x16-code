FP math_vecx ( unsigned char  angle,
char  speed 
)
inline

Calculates the x vector from an angle and speed.

In equinoxe world, an angle is not 360 degrees but 64 degrees to complete a round circle! Each quadrant is subdivided in 16 degrees. The x/y vectors are calculated from a common cos/sin table. The 0° vector is on the x axis and the 16° vector is on the y axis. Rotation is done clockwise. This brings us to the following table: >= < dx dy 0 16 1 1 16 32 -1 1 32 48 1 -1 48 64 -1 -1

The speed dx and dy are calculated taking into account the required speed.
