◆ readJoy1()

char readJoy1 ( )

Read Standard Controller #1 Returns a byte representing the pushed buttons

  • bit 0: right
  • bit 1: left
  • bit 2: down
  • bit 3: up
  • bit 4: start
  • bit 5: select
  • bit 6: B
  • bit 7: A
