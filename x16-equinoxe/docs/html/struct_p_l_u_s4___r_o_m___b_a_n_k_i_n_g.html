Detailed Description

ROM configuration for the machine, which normally has the BASIC and kernal ROMs enabled. The ROM configuration is adjusted by writing to the registers (the value is irrelevant). The upper portion of the kernal ROM at $FC00-$FCFF is always enabled no matter what the memory configuration, as are the I/O registers.
