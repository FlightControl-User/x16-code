void dtvSetCpuBankSegment1 ( char  cpuBankIdx)

Set the memory pointed to by CPU BANK 1 SEGMENT ($4000-$7fff) This sets which actual memory is addressed when the CPU reads/writes to $4000-$7fff The actual memory addressed will be $4000*cpuSegmentIdx
