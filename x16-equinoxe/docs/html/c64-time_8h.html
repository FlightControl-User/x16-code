◆ CLOCKS_PER_INIT

const unsigned long CLOCKS_PER_INIT = 0x12

Clock cycles used to start & read the cycle clock by calling clock_start() and clock() once. Can be subtracted when calculating the number of cycles used by a routine. To make precise cycle measurements interrupts and the display must be disabled so neither steals any cycles from the code.
