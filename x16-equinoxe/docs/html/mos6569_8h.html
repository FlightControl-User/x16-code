$D011 Control Register #1

  • Bit#0-#2: YSCROLL Screen Soft Scroll Vertical
  • Bit#3: RSEL Switch betweem 25 or 24 visible rows RSEL| Display window height | First line | Last line -—+-----------------------—+----------—+-------— 0 | 24 text lines/192 pixels | 55 ($37) | 246 ($f6) 1 | 25 text lines/200 pixels | 51 ($33) | 250 ($fa)
  • Bit#4: DEN Switch VIC-II output on/off
  • Bit#5: BMM Turn Bitmap Mode on/off
  • Bit#6: ECM Turn Extended Color Mode on/off
  • Bit#7: RST8 9th Bit for $D012 Rasterline counter Initial Value: %10011011
