Allocated a specified size of memory on the heap of the segment.

Parameters
segmentThe segment identifier, a value between 0 and 15.
sizeSpecifies the size of memory to be allocated. Note that the size is aligned to an 8 byte boundary by the memory manager. When the size of the memory block is enquired, an 8 byte aligned value will be returned.
Returns
